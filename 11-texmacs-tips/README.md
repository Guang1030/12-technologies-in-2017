# 11 TeXmacs Tips
# Tip 1: 中文标点和拼音的Tab变换
```
a tab → ā
a tab tab → á
a tab tab tab → ǎ
a tab tab tab tab → à

< tab → 《
[ tab → 「
[ tab tab → 『
‘ tab → ’
’ tab → ‘
" tab → “
" tab tab → ” 
```

# Tip 2: Convert to PDF using commandline
